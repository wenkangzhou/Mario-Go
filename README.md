# Super Mario Bros. 1-1 (Godot 4 复刻)

一个使用 Godot 4 + GDScript 实现的经典《超级马里奥兄弟》1-1 关卡风格的横版平台跳跃游戏。所有游戏节点均通过代码动态创建，关卡地图以字符矩阵形式定义，游戏素材使用 Python Pillow 程序生成。

## 技术栈

- **引擎**: Godot 4.x
- **语言**: GDScript
- **素材生成**: Python 3 + Pillow
- **开发方式**: 纯代码驱动，无 GUI 节点拖拽

## 游戏特性

### 核心操作
- 左右移动、跳跃（支持加速跑更高更远）
- 蹲下（可进入传送水管）
- 发射火球（火焰马里奥状态）

### 成长系统
- **小马里奥** → 吃蘑菇变 **大马里奥**
- **大马里奥** → 吃火焰花变 **火焰马里奥**（可发射火球）
- 受伤会降级，小马里奥受伤则失去生命

### 道具与物品
- **蘑菇**: 成长 / 1UP（绿蘑菇加命）
- **火焰花**: 获得射击能力
- **无敌星**: 10 秒无敌，可撞死敌人
- **金币**: 收集 100 枚奖励额外生命
- **隐藏砖块**: 顶出金币、道具
- **问号砖块**: 可顶出道具或连续金币

### 关卡元素
- 水管（含传送水管，蹲下进入可加命）
- 食人花（从水管中探出）
- 砖块与多层平台
- 悬崖陷阱
- 终点旗杆与城堡（到达后放烟花庆祝）

### 敌人 AI
- **板栗仔（Goomba）**: 巡逻移动，掉落悬崖会转向
- **乌龟（Koopa）**: 行走、缩壳、滑踢三连机制
  - 踩一下变静止龟壳
  - 再踩一下停止滑动的龟壳
  - 侧面触碰静止龟壳可踢飞
- 敌人之间会互相避让，滑动龟壳可撞死其他敌人

### 计分与连击
- 连续踩敌人可触发连击倍率（100 → 200 → 400 → ... → 1UP）
- 通关时剩余时间会折算为额外分数

## 如何运行

### 1. 生成游戏素材

```bash
python3 gen_sprites.py
```

这会生成所有 `*.png` 素材文件。

### 2. 运行游戏

将项目文件夹作为 Godot 4 项目打开，运行主场景即可。

> 注意：部分音效文件（`jump.wav`、`coin.wav`、`stomp.wav`、`hurt.wav`、`win.wav`、`pipe.wav`、`bgm.wav`）需要自行准备并放入项目根目录。

## 操作说明

| 按键 | 功能 |
|------|------|
| `←` / `→` | 左右移动 |
| `Space` | 跳跃 |
| `↓` | 蹲下（站在传送水管上可进入） |
| `Shift` / `B` | 加速跑（跳得更高更远） |
| `Z` | 发射火球（需火焰马里奥状态） |
| `Esc` | 暂停/继续 |

## 实现亮点

- **纯代码节点创建**: 所有 `CharacterBody2D`、`Area2D`、`Sprite2D`、`CollisionShape2D` 等均通过 `.new()` 动态实例化并组装，不依赖 Godot 编辑器场景树
- **字符地图驱动关卡**: 16×160 的字符矩阵定义整个关卡布局，便于快速迭代调整
- **像素网格画板**: 使用 `draw_grid()` 函数将字符矩阵渲染为 NES 风格的像素角色，支持多尺寸画布居中适配
- **双端碰撞处理**: 玩家与敌人的碰撞由玩家脚本主导，敌人脚本只处理非玩家交互，避免逻辑冲突
- **Tween 动画系统**: 成长闪烁、无敌星变色、敌人死亡弹飞、火球抛物线、烟花爆炸等效果均使用 Godot Tween 实现
