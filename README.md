# 流浪的夏天 / Wandering Summer (Godot 4)

一个使用 Godot 4 + GDScript 实现的横版平台跳跃游戏。1-1 关卡致敬经典《超级马里奥兄弟》风格，1-2 关卡则是完全独立的夏日天台冒险——两种美术风格、两套主角、两个世界，通过水管无缝切换。

所有游戏节点均通过代码动态创建，关卡地图以字符矩阵形式定义，游戏素材使用 Python Pillow 程序生成。

## 技术栈

- **引擎**: Godot 4.x
- **语言**: GDScript
- **素材生成**: Python 3 + Pillow
- **开发方式**: 纯代码驱动，无 GUI 节点拖拽

## 游戏特性

### 关卡

#### 1-1 — 经典复刻
- Goomba、Koopa、砖块、问号砖、蘑菇、星星、火焰花
- 食人花、云朵、城堡、终点旗杆（到达后放烟花）
- 传送水管可进入 1-2

#### 1-2 — 流浪的夏天
- **主角「小夏」**: 红衣高中生、黑短发、背书包
- **天台场景**: 南方小城天台日落，水泥平台、空调外机、铁栅栏、晾衣绳
- **流浪猫敌人**: 巡逻 / 警觉 / 追逐 / 扑击 / 受伤 完整状态机
- **夏日收集物**: 西瓜、冰棍、磁带，拾取漂浮动画
- **检查点**: 天台红旗记录进度
- **冷却塔终点**: 到达后触发通关
- **移动平台**: 可站立的 AnimatableBody2D

### 核心操作
- 左右移动、跳跃（支持加速跑更高更远）
- 蹲下（可进入传送水管）
- 发射火球（火焰马里奥状态，仅限 1-1）

### 成长系统（1-1）
- **小马里奥** → 吃蘑菇变 **大马里奥**
- **大马里奥** → 吃火焰花变 **火焰马里奥**（可发射火球）
- 受伤会降级，小马里奥受伤则失去生命

### 道具与物品
- **蘑菇**: 成长 / 1UP（绿蘑菇加命）
- **火焰花**: 获得射击能力
- **无敌星**: 10 秒无敌，可撞死敌人
- **金币**: 收集 100 枚奖励额外生命
- **隐藏砖块**: 顶出金币、道具
- **问号砖块**: 可顶出道具或连续金币

### 画面表现
- 标题画面淡入 + 闪烁提示
- 屏幕震动、受伤红闪
- 落地扬尘 / 跑步扬尘粒子
- 无敌星 Tween 变色
- 敌人死亡弹飞、火球抛物线、烟花爆炸

### 敌人 AI
- **板栗仔（Goomba）**: 巡逻移动，掉落悬崖会转向
- **乌龟（Koopa）**: 行走、缩壳、滑踢三连机制
  - 踩一下变静止龟壳
  - 再踩一下停止滑动的龟壳
  - 侧面触碰静止龟壳可踢飞
- **流浪猫**: 天台边缘巡逻，发现玩家后追逐，近身扑击，可被踩头消灭
- 敌人之间会互相避让，滑动龟壳可撞死其他敌人

### 计分与连击
- 连续踩敌人可触发连击倍率（100 → 200 → 400 → ... → 1UP）
- 通关时剩余时间会折算为额外分数

```
├── brick.gd              # 砖块逻辑
├── question_block.gd     # 问号砖/连续砖逻辑
├── coin.gd               # 金币逻辑
├── mushroom.gd           # 蘑菇逻辑
├── flower.gd             # 火焰花逻辑
├── star.gd               # 无敌星逻辑
├── collectible.gd        # 夏日收集物（西瓜/冰棍/磁带）逻辑
├── fireball.gd           # 火球逻辑
├── piranha.gd            # 食人花逻辑

├── gen_sprites.py        # Python 素材生成脚本（含 SMB1 + 夏日主题）
├── gen_sfx.py            # Python 音效/BGM 生成脚本
└── *.png / *.wav         # 游戏素材与音效
```

## 如何运行

### 1. 生成游戏素材与音效

```bash
python3 gen_sprites.py
python3 gen_sfx.py
```

这会生成所有 `*.png` 素材文件和 `*.wav` 音效文件。

### 2. 运行游戏

将项目文件夹作为 Godot 4 项目打开，运行主场景即可。

## 操作说明

| 按键 | 功能 |
|------|------|
| `←` / `→` | 左右移动 |
| `Space` | 跳跃 |
| `↓` | 蹲下（站在传送水管上可进入 1-2） |
| `Shift` / `B` | 加速跑（跳得更高更远） |
| `Z` | 发射火球（需火焰马里奥状态） |
| `Esc` | 暂停/继续 |

## 实现亮点

- **纯代码节点创建**: 所有 `CharacterBody2D`、`Area2D`、`Sprite2D`、`CollisionShape2D` 等均通过 `.new()` 动态实例化并组装，不依赖 Godot 编辑器场景树
- **字符地图驱动关卡**: 16×160（1-1）/ 16×80（1-2）的字符矩阵定义关卡布局，便于快速迭代调整
- **双主角系统**: 同一份 `player.gd` 根据 `current_level` 自动切换马里奥/小夏的动画、纹理与碰撞尺寸
- **像素网格画板**: 使用 `draw_grid()` 函数将字符矩阵渲染为 NES 风格的像素角色，支持多尺寸画布居中适配
- **双端碰撞处理**: 玩家与敌人的碰撞由玩家脚本主导，敌人脚本只处理非玩家交互，避免逻辑冲突
- **Tween 动画系统**: 成长闪烁、无敌星变色、敌人死亡弹飞、火球抛物线、烟花爆炸、标题淡入等效果均使用 Godot Tween 实现
